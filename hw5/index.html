Ionic

Pros: Ionic is an open source, hybrid app development software allowing the user to choose any user interface framework they want, such as Angular, React or Vue.js.

Cons: Ionic is not backed by any major company like Google's Flutter or Facebooks' React Native. It is also under the MIT license, not the GNU license, even though it is open source. Finally, on VS Code's extension website it says Ionic has been deprecated in favor of the WebNative.dev extension.
